## Install

e.g: use `@eslint-sets/eslint-config`

```shell
# by npm
npm install -D @eslint-sets/eslint-config
# by yarn
yarn add @eslint-sets/eslint-config
# by pnpm
pnpm install @eslint-sets/eslint-config
```

## Usage

```js
// .eslintrc.js
module.exports = {
  extends: ['@eslint-sets'],
  rules: {
    // custom rules
  }
}
```
