# 框架配置

本节提供框架特定的配置示例。

## 概述

配置支持自动检测多个框架，也支持显式配置。

## 可用框架

- TypeScript: `typescript: true`
- Vue: `vue: true` 或 `vue: { vueVersion: 2|3 }`
- React: `react: true`
- Svelte: `svelte: true`
- Next.js: `nextjs: true`
- Nuxt: `nuxt: true`
- Angular: `angular: true`
- Astro: `astro: true`
- UnoCSS: `unocss: true`

详细配置请参考英文文档或各框架特定指南。
