# 进阶用法

详细的进阶配置请参考 [英文版本](/guide/advanced-usage)。

## 主要内容

### TypeScript 高级选项

- 类型感知规则（typeAware）
- 可擦除语法（erasableOnly）

### 外部格式化器

支持 CSS、HTML、Markdown 等文件的 Prettier 格式化。

### 独立配置导入

可单独导入各个配置模块组合使用。

### 默认规则行为

- 无行长度限制
- 使用全局 Buffer/process
- 灵活的未使用表达式
- 混合运算符分组

### 框架特定功能

- React Compiler 支持
- React Server Components
- Vue 版本控制
- Svelte 5 Runes

详细配置和示例请查看英文文档。
