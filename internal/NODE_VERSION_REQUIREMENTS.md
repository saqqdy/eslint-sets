# ESLint 插件 Node.js 版本要求

本文档记录了 `@eslint-sets/eslint-config` 项目中所有 ESLint 插件对各版本 Node.js 的要求。

## 最低 Node.js 版本要求汇总

### 方案一：使用最新版本

**最低 Node.js: `20.19.0` 或 `22.13.0`**

受限于以下插件：
- `eslint@9`
- `eslint-plugin-regexp@3.x`
- `eslint-plugin-jsdoc@62.x`
- `eslint-plugin-toml@1.x`

### 方案二：降级部分插件版本

**最低 Node.js: `18.18.0`**

需要降级的插件：

| 插件 | 从 | 降到 |
|------|-----|-----|
| `eslint-plugin-regexp` | ^3.1.0 | ^2.10.0 |
| `eslint-plugin-jsdoc` | ^62.8.0 | ^61.7.1 |
| `eslint-plugin-perfectionist` | ^5.6.0 | ^4.15.1 |
| `eslint-plugin-unicorn` | ^63.0.0 | ^59.0.1 |
| `eslint-plugin-svelte` | ^3.0.0 | ^2.46.1 |
| `eslint-plugin-vue` | ^10.0.0 | ^9.33.0 |

### 特殊说明

- `eslint-plugin-toml@1.x` 要求 Node.js `^20.19.0 || ^22.13.0 || >=24`，如需支持 Node.js 18 可降级到 `^0.13.1`
- `eslint-plugin-unicorn@56.x` 兼容所有 ESLint 9.x 版本（不仅限于 9.22.0+）
- `eslint-plugin-jsonc@3.x` 和 `eslint-plugin-yml@3.x` 同样要求更高版本 Node.js，建议保持 2.x 和 1.x 版本
