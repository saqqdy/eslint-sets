## Install

```bash
# use npm
$ npm install @eslint-sets/egg --save

# use yarn
$ yarn add @eslint-sets/egg

# use pnpm
$ pnpm install @eslint-sets/egg
```

## Usage

```js
const { nextIndex } = require('@eslint-sets/egg')

nextIndex()
```
