## Install

```bash
# use pnpm
$ pnpm install -D @eslint-sets/eslint-config-vue3

# use npm
$ npm install -D @eslint-sets/eslint-config-vue3

# use yarn
$ yarn add -D @eslint-sets/eslint-config-vue3
```

## Usage

```js
// .eslintrc.js
module.exports = {
  extends: ['@eslint-sets/vue3'],
  rules: {
    // custom rules
  }
}
```
